You can install the dotfiles by running following commands:

```bash
git clone --recurse-submodules git@github.com:RobinVdBroeck/dotfiles.git
cd dotfiles
./install.sh
```

## Supports:
1. zsh (auto install oh-my-zsh)
2. nvim
3. git
4. tmux (+ tmuxp)
5. Alacritty
6. scripts (usefull scripts that I use)
